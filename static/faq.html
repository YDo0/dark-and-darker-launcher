F.A.Q

When can I run the game?

You need to download Installer and required hotfix. Start main game installer and the highest # patch.
Then "Play game" button will be able to start Dark and Darker.

How can I check for updates?

Just go to File - Check for game updates
It should automatically sync all the new patches.

Where can I limit download / upload speed?

Go to Edit - Preferences
Use checkboxes to enable download / upload speed limits.

Can I close the launcher after I got the game?

Sure! But I would recommend keep it opened or keep it in tray to continue seeding files for the other players.
